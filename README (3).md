# MANDALART DRAFT — マンダラチャート設計ツール

9×9のマンダラチャートをブラウザ上で作成・編集し、AIへの読み込みや資料化を前提に
PNG／テキストの両形式で出力できる単体HTMLツールです。外部AI APIは使用していません。

---

## 1. できること

### マンダラ本体（9×9グリッド）
- 中央ブロック中心＝**最終目標**
- 中央ブロック周囲8マス＝**サブテーマ**（8個）
- 各サブテーマは、対応する外周ブロックの中心セルに自動複製されます（どちらを編集しても同期）
- 各外周ブロックの残り8マス＝そのサブテーマの**具体的アクション**（合計64個）

### アイディア・スロット（強制結合）
- 「🎲 シャッフル」を押すと、グリッド内の入力済みマスからランダムに2つを抽出し
  「[A] × [B] で何かできないか？」という組み合わせを提示
- 3スロット分保存可能。各スロットに浮かんだアイディアを自由記述できます
- 意図しない組み合わせから新結合（セレンディピティ）を狙う機能です

### アクション・コンバーター（Todo変換）
- 各マス右上の「☆」をクリックしてアイディアをマーク（★になります）
- 「＋ テーマ追加」で複数のTodoテーマ（リスト）を作成可能
- 各テーマの「マーク済みを変換」で、★マークされたマスを一括でTodo項目化
- 「＋ 項目」で手動追加、チェックボックスで完了管理も可能

### 出力機能
| 機能 | 内容 |
|---|---|
| PNG出力 | グリッド＋タイトルブロックを画像化してダウンロード（`mandalart.png`） |
| テキスト出力 | マンダラ本体・アイディアスロット・Todoリストを構造化Markdownとして書き出し（`mandalart.txt`）。AIへの貼り付け・読み込みを想定した形式です |

---

## 2. 使い方

1. `mandalart.html` をブラウザで開く（ダブルクリックでOK。サーバー不要）
2. 「サンプル読込」で構造を確認、または直接各マスをクリックして入力
3. アイディアが欲しいときはアイディア・スロットの「🎲シャッフル」を活用
4. 良いアイディアのマスは「☆」でマークし、テーマごとにTodo化
5. 「PNG出力」「テキスト出力」で必要な形式を書き出し

---

## 3. データの取り扱い

- 入力内容はブラウザのメモリ上にのみ保持されます（`localStorage`等は不使用）
- **ページをリロードすると内容は消去されます。** 作業を残したい場合は都度「テキスト出力」でバックアップしてください
- 通信は行わず、PNG生成用ライブラリ（html2canvas）以外の外部送信は一切ありません

---

## 4. ファイル構成

```
mandalart.html   … 本体（HTML/CSS/JSを1ファイルに集約）
README.md        … 本ドキュメント
```

---

## 5. 技術メモ

- 依存ライブラリ：`html2canvas`（PNG出力用、CDN読み込み）
- AI API・外部通信：不使用
- グリッド構造はDOM上で `data-block` / `data-cell` / `data-role`（main / sub / subcenter / action）属性により管理
- サブテーマの同期はイベントリスナーによるDOM間ミラーリング方式
